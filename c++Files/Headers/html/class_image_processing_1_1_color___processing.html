Detailed Description

A class that helps to extract information about the color values of pixels in an image.

This class helps generating a color filter given an image. It will run through the pixels of an input image and store the highest and lowest HSV values. Given that information it creates a filter for that color range. Also it is possible to construct a black filter, meaning to get the lowest SV values of an Image. This way it is possible to extract dark regions of an image
